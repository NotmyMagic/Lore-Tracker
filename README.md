# capstone-project

A Website to keep track of and Edit D&D and other role playing games. You will be able to write down lore and categorize it into different categories. Create world reigions, character backgrounds, custom spells, custom feats, and custom Items. Keep a spoiler wall for future sessions.

Full functionality on the website a dm can keep track of lore information related to their worlds. This tool will help dms to organize their worlds and lore allowing them to have easy access incase their players ever have any questions
